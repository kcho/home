#How to sort data frame ## using 'order()' fucntion <br> <br> <br> ``` # by column names prac[with(prac,order(Subject.name)),] prac[with(prac,order(-Subject.name)),] #descending order # by column numbers prac[order(prac[,1])] prac[order(-prac[,1])] #descending order # send option to add prac[with(prac,order(Subject.name,group)),] ```
